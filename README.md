# SharpEdge Sports Analytics

## Overview
SharpEdge is a high-performance sports analytics dashboard designed to identify betting edges with a target win rate of >65%.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Vanilla CSS (Variables + Glassmorphism)
- **Language**: TypeScript
- **Icons**: Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)
